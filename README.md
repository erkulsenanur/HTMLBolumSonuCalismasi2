

# HTML Bölüm Sonu Çalışması
Bu repo [Kodluyoruz](https://www.kodluyoruz.org/) Front-End Eğitiminde oluşturduğum HTML bölüm sonu çalışmasını içermektedir. İçerisinde bir adet README dosyası, bir adet index.html, bir adet contact.html ve bir adet about-us.html dosyaları ile beraber img klasörünü barındırıyor.

## Installation
Öncelikle projeyi clonelayın.
```
git clone https://github.com/erkulsenanur/HTMLBolumSonuCalismasi2.git
```

## Usage
Projeyi klonladıktan sonra Visual Studio Code programında açınız.

Linux için:
```
cd HTMLBolumSonuCalismasi2
code .
```
